# Capycopter

## Компоненты

Это будет гоночный дрон. Пропеллер 5 дюймов. На ручном управлении. Взлётнтый вес 500г. Рама из PETG. Управление: ExpressLRS. Видео аналоговое.Диагональ 220 милиметров. Сумарнная тяга мотора 2кг.

- моторчики 4 шт,  5'' - 2207/2306
- пропеллеры 5x4, 3x3
- корпус резиновые демпферы
- пульт ЕxpressLRS 2.4GHz
- батарея - 1300-1500mAh,4s
- антенна
  - elrs: whip
  - 5.8GHz, RHSP, 50Om, разьём sma9(vdx)
- контроллер - F7, гираскоп ASM/MPU,прошивка-betaflight
- камера - аналоговая 5.8GHz,VTX600-800mW(runcam phoenix)
- зарядка для акума потддержка 4S 1-5A,storage mode
- fpv очки
  - diversiti receiver
  - DVR
- желание
- ESC 35-45A
- LiPo-save мешок
