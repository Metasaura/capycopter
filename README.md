# Capycopter

## Компоненты

Это будет гоночный дрон.Пропелер 5 дюймов.На ручном управлений.Взолётнтый вес 500г.Рама из PETG.Управление:ExpressLRS.Видио аналаговае.Диоганаль 220млиметров.Сумарная тяга мотора 2кг.

- моторчики 4 шт,  5'' - 2207/2306
- пропелеры 5x4,3x3
- корпус резинавые демпферы
- пульт xpresslrs 2.4GHz
- батарея - 1300-1500mAh,4s
- антенnа
  - elrs:whip
  - 5.8GHz,RHSP,50Om,разьёмsma9(vdx)
- контроллер - F7,гераскоп ASM/MPU,прошивка-betaflight
- камера - аналагавая 5.8GHz,VTX600-800mW(runcam phoenix)
- зарядка для акума подтержлка 4S 1-5A,storage mode
- fpv очки
  - doversiti receiver
  - DVR
- желание
- ESC 35-45A
- LiPo-save мешок
